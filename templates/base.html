About This Project

This Inventory Management System is a Flask-based web application designed to help businesses manage products, track stock levels, monitor movements, and generate reports for better decision-making.

Workflow

  1. Create Products: Define items like Product A, Product B, etc.
  2. Set up Locations: Add warehouses or storage areas like Location X, Y, Z.
  3. Record Movements: Track product transfers between locations (e.g., Product A → Location X).
  4. Generate Reports: View stock balances per product and location in real-time.

Key Features

  • Simple product and warehouse management
  • Real-time tracking of stock movements
  • Inventory balance reports with product-wise and location-wise breakdowns
  • Responsive and user-friendly design

Modules

Product Management

Add, update, and view all available products.

Location Management

Manage multiple warehouses and stock points.

Movements
